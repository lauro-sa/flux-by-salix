---
name: No preguntar cómo seguir, hacer lo que pide
description: Ejecutar lo que el usuario pide sin preguntar si quiere seguir o hacer otra cosa primero
type: feedback
---

No preguntar "¿querés que lo haga ahora o primero hacemos X?". Si el usuario pide algo, hacerlo directamente.

**Why:** El usuario se frustra cuando le pregunto cómo seguir en vez de ejecutar. Él decide la prioridad, yo ejecuto.

**How to apply:** Cuando el usuario pide una mejora o cambio, implementarlo inmediatamente sin ofrecer alternativas de orden.
